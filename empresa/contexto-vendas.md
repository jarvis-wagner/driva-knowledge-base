# Driva - Contexto para Vendas

> Este arquivo deve ser usado como contexto em TODOS os prompts de geração de copy e abordagem comercial.

## Posicionamento

**A Driva é uma plataforma completa de Go-To-Market (GTM) focada em Inteligência de Mercado.**

Não somos apenas uma base de dados ou uma ferramenta de prospecção. Somos a plataforma que conecta inteligência de mercado, dados e automação para empresas venderem mais e melhor.

## O que a Driva oferece

### 1. Inteligência de Mercado
- **Mapeamento de TAM/SAM/SOM** — Entenda o tamanho real do seu mercado
- **Análise de penetração** — Saiba onde você já está e onde pode crescer
- **Identificação de ICP** — Descubra o perfil ideal de cliente com dados reais
- **Dashboards personalizados** — Visualize oportunidades de forma estratégica

### 2. Dados Exclusivos (Datapacks)
Mais de **300 bases de dados** proprietárias que vão muito além do CNPJ básico:
- Frotas, Energia, Fiscal, E-commerce, Food Service, Agro, Saúde Animal, etc.
- Dados que permitem segmentação e análise que nenhum concorrente oferece

### 3. Geração de Leads Qualificados
- Criação de listas segmentadas com filtros avançados
- Enriquecimento de dados (telefones, emails, decisores)
- Integração direta com CRM

### 4. Engajamento Multicanal
- Prospecção automatizada via LinkedIn, WhatsApp e Email
- Cadências personalizadas
- IA para otimização de abordagem

### 5. Driva Copilot
- Assistente de IA via WhatsApp
- Dossiês de prospects em segundos
- Preparação automática para reuniões

## Proposta de Valor Central

**"Transformamos dados em estratégia de vendas."**

A Driva não vende dados soltos. Vendemos:
- **Clareza** sobre onde o cliente pode crescer
- **Precisão** na identificação de oportunidades
- **Eficiência** no processo de prospecção
- **Inteligência** para decisões estratégicas

## Diferenciais vs Concorrentes

| Driva | Concorrentes típicos |
|-------|---------------------|
| Plataforma GTM completa | Ferramentas pontuais |
| 300+ bases exclusivas | Dados genéricos de CNPJ |
| Inteligência de mercado | Apenas listas de leads |
| TAM/SAM/SOM integrado | Sem visão estratégica |
| Onboarding consultivo | Self-service básico |

## Tom de Comunicação

Quando abordamos um prospect:
1. **Consultivo** — Entendemos o negócio antes de oferecer produto
2. **Estratégico** — Falamos de crescimento, não de features
3. **Data-driven** — Usamos dados para justificar a proposta
4. **Direto** — Sem enrolação, foco em valor

# Obras

## Descrição Geral
Obras - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 39

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| CNO | Cadastro Nacional de Obras - Número de matrícula da obra |
| NOME | Nome ou identificação da obra |
| CÓDIGO DO PAIS | Código do país onde a obra está localizada |
| NOME DO PAIS | Nome do país onde a obra está localizada |
| DATA DE INÍCIO | Data de início da obra |
| DATA DE INICIO DA RESPONSABILIDADE | Data de início da responsabilidade pela obra |
| DATA DE REGISTRO | Data de registro da obra no CNO |
| CNO VINCULADO | CNO de obra vinculada |
| CEP | CEP do endereço da obra |

*E mais 30 campos adicionais...*

## Casos de Uso Práticos
Este datapack pode ser utilizado para:
- Enriquecimento de bases de dados existentes
- Análise de mercado e inteligência comercial
- Segmentação e targeting de clientes
- Due diligence e análise de risco

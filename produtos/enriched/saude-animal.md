# Saude Animal

## Descrição Geral
Saúde Animal - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 64

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| CRMV | Número de registro no Conselho Regional de Medicina Veterinária. |
| UF DO CRMV | Estado de registro no CRMV. |
| SITUAÇÃO DO REGISTRO | Status do registro: Ativo ou Não ativo. |
| ÁREAS DE ATUAÇÃO | Atividades da empresa: Pet Shop, Clínica, Hospital, Agropecuária, Laboratório, etc. |
| VETERINÁRIO RESPONSÁVEL | Nome do profissional responsável técnico. |
| CRMV DO RESPONSÁVEL | Número do CRMV do veterinário responsável. |
| UF DO CRMV DO RESPONSÁVEL | Estado do registro do veterinário. |
| CLASSE DO PROFISSIONAL | Tipo de profissional: VP (Veterinário Principal), ZP (Zootecnista Principal), VS, ZS. |
| DOMICÍLIOS COM CACHORRO | Quantidade estimada de domicílios com cães. |

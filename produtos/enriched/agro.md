# Agro

## Descrição Geral
Agro - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 59

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| CÓDIGO CAR | Código do Cadastro Ambiental Rural (identificador único da propriedade). |
| ÁREA TOTAL (HA) | Área total da propriedade em hectares. |
| MÓDULOS FISCAIS | Quantidade de módulos fiscais da propriedade. |
| TIPO DE IMÓVEL | Classificação: IRU (Imóvel Rural), AST (Assentamento), PCT (Comunidade Tradicional). |
| STATUS DO CAR | Status do cadastro: AT (Ativo), PE (Pendente), SU (Suspenso), CA (Cancelado). |
| SITUAÇÃO AMBIENTAL | Situação normalizada: Regular, Aguardando análise ou Cancelado. |
| CONDIÇÃO DETALHADA | Descrição detalhada da situação do CAR junto ao órgão ambiental. |
| DATA DE CADASTRO | Data de criação do CAR. |
| DATA DE ATUALIZAÇÃO | Data da última atualização do CAR. |

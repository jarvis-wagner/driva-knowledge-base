# Foodservice

## Descrição Geral
Food Service - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 61

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| SEGMENTAÇÃO | Segmentação do estabelecimento (Bares e Restaurantes/Padaria/Cafeteria/etc). |
| SUBSEGMENTAÇÃO | Subsegmentação detalhada do estabelecimento. |
| TIPOLOGIA | Tipo de culinária ou especialidade (Hambúrguer/Pizza/Japonesa/Brasileira/etc). |
| FAIXA DE PREÇO | Faixa de preço do estabelecimento ($ a $$$$). |
| CLASSIFICAÇÃO | Classificação do estabelecimento por porte (A/B/C/D). |
| OPERADORES DE CAIXA | Quantidade estimada de operadores de caixa (proxy de porte). |
| FAZ DELIVERY | Indica se o estabelecimento faz delivery. |
| FAZ RETIRADA | Indica se aceita retirada no local (takeaway). |
| CONSUMO LOCAL | Indica se tem mesas para consumo no local. |

# Residuos

## Descrição Geral
Resíduos - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 116

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| NÚMERO CDF SINIR | Número do Certificado de Destinação Final de resíduos. |
| DATA EMISSÃO CDF | Data de emissão do certificado de destinação. |
| CNPJ GERADOR RESÍDUO | CNPJ da empresa geradora do resíduo. |
| MUNICÍPIO GERADOR | Município onde o resíduo foi gerado. |
| UF GERADOR | Estado onde o resíduo foi gerado. |
| CNPJ DESTINATÁRIO RESÍDUO | CNPJ da empresa receptora/tratadora do resíduo. |
| UNIDADE DESTINATÁRIO | Unidade/filial que recebeu o resíduo (cidade - UF). |
| CÓDIGO E DESCRIÇÃO RESÍDUO | Código IBAMA + descrição do resíduo (ex: 160117 - Sucatas metálicas ferrosas). |
| CLASSE RESÍDUO SINIR | Classificação ABNT (Classe I - Perigoso / Classe II A - Não Inerte / Classe II B - Inerte). |

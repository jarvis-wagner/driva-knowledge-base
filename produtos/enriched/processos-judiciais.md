# Processos Judiciais

## Descrição Geral
Processos Judiciais - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 119

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| NÚMERO DO PROCESSO | Número identificador único do processo judicial |
| ÁREA | Área do direito (cível, trabalhista, criminal, etc) |
| DATA DE AUTUAÇÃO | Data em que o processo foi registrado no sistema |
| DATA DE DISTRIBUIÇÃO | Data em que o processo foi distribuído ao juiz |
| É JUSTIÇA GRATUITA | Indica se a parte tem benefício de justiça gratuita |
| É PRIORITÁRIO | Indica se o processo tem tramitação prioritária |
| É PROCESSO DIGITAL | Indica se o processo tramita eletronicamente |
| É SEGREDO DE JUSTIÇA | Indica se o processo corre em segredo de justiça |
| TEM TUTELA ANTECIPADA | Indica se há pedido de tutela antecipada |

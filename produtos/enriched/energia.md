# Energia

## Descrição Geral
Energia - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 39

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| NÍVEL DE TENSÃO | Tensão de fornecimento: Alta (≥69kV), Média (1-69kV) ou Baixa (<1kV). |
| TENSÃO DE FORNECIMENTO | Tensão nominal do ponto de conexão em kilovolts (KV). |
| GRUPO TARIFÁRIO | Modalidade tarifária contratada: A1, A2, A3, A4, B1, B2, B3 ou B4. |
| MODALIDADE TARIFÁRIA | Tipo de contrato de energia: convencional ou horária. |
| CLASSE DE CONSUMO | Classificação da unidade: Comercial, Industrial, Poder Público, Rural ou Serviço Público. |
| DATA DE CONEXÃO | Data em que a empresa foi conectada à rede elétrica. |
| CARGA INSTALADA | Potência elétrica total instalada no estabelecimento (KW). |
| DEMANDA CONTRATADA | Demanda de potência contratada com a distribuidora (KW). |
| DEMANDA PONTA | Maior demanda registrada no horário de ponta do mês (KW). |

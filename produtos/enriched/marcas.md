# Marcas

## Descrição Geral
Marcas Registradas - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 3

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| POSSUI MARCA | Indica se a empresa possui alguma marca registrada. |
| QTDE MARCAS | Quantidade de marcas registradas da empresa. |
| MARCAS REGISTRADAS | Lista de marcas registradas pela empresa. |

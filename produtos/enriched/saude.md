# Saude

## Descrição Geral
Saude - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 70

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| Nome | Nome completo do profissional de saúde |
| Nome social | Nome social (se houver) |
| Número do CRM | Número de registro no Conselho Regional de Medicina |
| UF do registro | Estado do registro no conselho |
| Município | Município de atuação |
| Especialidade | Especialidade médica registrada |
| Data de graduação | Data em que se formou |
| Instituição de graduação | Faculdade onde se graduou |
| Tem pós-graduação | Indica se possui pós-graduação |

*E mais 61 campos adicionais...*

## Casos de Uso Práticos
Este datapack pode ser utilizado para:
- Enriquecimento de bases de dados existentes
- Análise de mercado e inteligência comercial
- Segmentação e targeting de clientes
- Due diligence e análise de risco

## Público-Alvo
- Empresas de tecnologia
- Consultorias e agências
- Instituições financeiras
- Departamentos de vendas e marketing

## Integração com Outros Produtos
Este produto pode ser combinado com outros datapacks da Driva para análises mais abrangentes e insights mais profundos.

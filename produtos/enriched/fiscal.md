# Fiscal

## Descrição Geral
Fiscal - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 8

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| TEM PERDCOMP | Indica se possui pedido de restituição/compensação tributária. |
| NÚMERO PERDCOMP | Número do pedido de restituição/compensação. |
| TIPO DOCUMENTO PERDCOMP | Tipo de documento (restituição ou compensação). |
| TIPO CRÉDITO PERDCOMP | Tipo de crédito tributário solicitado. |
| DATA TRANSMISSÃO PERDCOMP | Data de transmissão do pedido. |
| DATA PROCESSAMENTO PERDCOMP | Data de processamento do pedido. |
| SITUAÇÃO PERDCOMP | Situação atual do pedido. |
| FOLHA SALARIAL ESTIMADA | Valor estimado da folha salarial mensal do grupo. |

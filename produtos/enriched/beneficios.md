# Beneficios

## Descrição Geral
Beneficios - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 38

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| POSSUI PAT | Indica se a empresa possui programa de alimentação do trabalhador. |
| QUANTIDADE BENEFICIADOS PAT | Número de beneficiados pelo programa de benefícios. |
| INSCRIÇÃO BENEFÍCIOS | Número de inscrição no programa de benefícios. |
| DATA DE INSCRIÇÃO | Data de inscrição no programa de benefícios. |
| STATUS BENEFÍCIOS | Status atual da inscrição no programa. |
| TOTAL TRABALHADORES BENEFICIADOS | Total de trabalhadores que recebem benefícios. |
| TRABALHADORES ATÉ 5 SALÁRIOS MÍNIMOS | Quantidade de trabalhadores beneficiados com até 5 salários mínimos. |
| TRABALHADORES ACIMA 5 SALÁRIOS MÍNIMOS | Quantidade de trabalhadores beneficiados acima de 5 salários mínimos. |
| TOTAL BENEFÍCIOS CONCEDIDOS | Total de benefícios concedidos pela empresa. |

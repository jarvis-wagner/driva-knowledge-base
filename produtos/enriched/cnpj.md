# Cnpj

## Descrição Geral
Dados Cadastrais - CNPJ - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 110

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| CNPJ | CNPJ completo do estabelecimento (14 dígitos). |
| RAIZ CNPJ | Raiz do CNPJ (8 primeiros dígitos) - identifica o grupo empresarial. |
| RAZÃO SOCIAL | Razão social da empresa. |
| NOME FANTASIA | Nome fantasia do estabelecimento. |
| MATRIZ | Indica se é matriz ou filial. |
| QTDE FILIAIS | Quantidade de filiais do grupo empresarial. |
| NATUREZA JURÍDICA | Natureza jurídica da empresa (LTDA, SA, MEI, etc). |
| CAPITAL SOCIAL | Capital social declarado da empresa. |
| PÚBLICA OU PRIVADA | Classificação se empresa é pública ou privada. |

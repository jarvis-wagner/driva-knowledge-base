# Ecommerce

## Descrição Geral
E-commerce - Datapack da Driva com dados estruturados para inteligência de mercado.

## Dados e Campos Disponíveis
Total de campos: 50

### Campos Principais

| Campo | Descrição |
|-------|----------|
| ------- | ---------- |
| DOMÍNIO | Endereço do site (ex: loja.com.br). |
| RAIZ CNPJ | Raiz do CNPJ vinculado ao domínio. |
| TECNOLOGIA | Tecnologia/plataforma detectada no site. |
| VISITAS | Quantidade estimada de visitas ao site. |
| TIPO DE TRÁFEGO | Classificação do tipo de tráfego (orgânico/pago). |
| CATEGORIA SIMILARWEB | Categoria do site classificada pela SimilarWeb. |
| PAÍS PRINCIPAL | País com maior percentual de acessos. |
| PERCENTUAL PAÍS | Percentual de acessos do país principal. |
| STATUS DO SITE | Indica se o site está online ou offline. |
